# Are-you-sick 你有病吗

默认初始密码：Abc123456

## 启动后端
cd *\Are you sick?\hospital\target

&&

java -jar HospitalApplication-0.0.1-SNAPSHOT.jar


或者：
cd *\Are you sick?\hospital

&&

mvn spring-boot:run

## 启动前端
cd *\Are you sick?\frontend

&&

npm run dev

## 访问
后端服务器: http://localhost:8080

前端服务器: http://localhost:5173
